# 📚 Library Management System

**Welcome to the Library Management System!** 🎉  
A **console-based Java application** designed to manage books and library members efficiently. Practice **OOP**, **ArrayLists**, and **Java programming** in a real-world scenario 💻✨.  

---

## 📝 Features

- **📖 Add Books** – Enter book details (Name, Author, Type, ID, Borrowed status)  
- **🔍 Search Books** – Find books by their ID  
- **📚🔄 Borrow & Return Books** – Update book status dynamically  
- **📖 Show All Books** – Display all books with full information  
- **👤 Create Membership** – Add library members with unique IDs  
- **🧑‍🤝‍🧑 Show Members** – Display member information  
- **🗃️ Dynamic Storage** – Uses `ArrayList` to store books and members  

---

## ⚙️ How It Works

1. Run the program in the console with a **menu interface**.  
2. Choose options by entering numbers corresponding to actions.  
3. Books and members are stored in **ArrayLists**, allowing dynamic addition & search.  
4. Borrowing a book updates its **borrowed status** automatically.  
5. Members can be added and displayed easily with unique IDs.  

---

## 💡 Key Concepts Used

- **🏗️ Object-Oriented Programming (OOP)** – Classes: `Book`, `Library`, `Member`  
- **🔒 Encapsulation** – Getters and setters for private variables  
- **📦 Collections** – `ArrayList` for dynamic data storage  
- **🔁 Loops & Conditional Statements** – Menu logic, searching, borrowing  
- **✍️ Scanner** – Interactive console input  

---

## 🎯 Why This Project is Useful
